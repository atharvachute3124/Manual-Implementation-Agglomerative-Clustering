## Overview
Manual Implementation of Agglomerative Clustering

### Dataset and required files
Uploaded file: "AllBooks_baseline_DTM_Labelled_modified.csv" or <br>
Download from here: https://archive.ics.uci.edu/ml/datasets/A+study+of++Asian+Religious+and+Biblical+Texts#

### Attribute Information
The attributes are just the words from the bag of words preprocessing of the mini-corpus made up of the 8 religious books considered in this study. There are 8265 words used.

### Dependencies:
pandas, numpy, sklearn

### How to execute
1. Run aggolmerative_clustering.py <br>
2. Results are saved in "agglomerative.txt" file as 8 seperate cluters having comma seperated data points.
